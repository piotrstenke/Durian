# DUR0306 - Error
### Friend type is specified multiple times by two different FriendClassAttributes.

## Example

### Code with diagnostic:
```csharp
using Durian;

// DUR0306
[FriendClass(typeof(Other))]
[FriendClass(typeof(Other))]
public class Test
{
	internal string Name { get; }
}

public class Other
{
}

```
