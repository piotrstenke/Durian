# DUR0304 - Error
### Type specified by a FriendClassAttribute cannot access the target type.

## Example

### Code with diagnostic:
```csharp
using Durian;

public class Test
{
    // DUR0304
    [FriendClass(typeof(Other))]
    private class Inner
    {
        internal static string Name { get; }
    }
}

public class Other
{
}

```
