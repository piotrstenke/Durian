# DUR0313 - Error
### Inner types don't need to be specified as friends explicitly.

## Example 1

### Code with diagnostic:
```csharp
using Durian;

// DUR0313
[FriendClass(typeof(Inner))]
public sealed class Test
{
	internal string Name { get; }
	
	public class Inner
	{
	}
}

```
