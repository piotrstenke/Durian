# DUR0302 - Error
### Member cannot be accessed outside of friend types.

## Example

### Code with diagnostic:
```csharp
using Durian;

[FriendClass(typeof(Other))]
public class Test
{
    internal static string Name { get; }
}

public class Another
{
    // DUR0302
    string name = Test.Name;
}

```
