# DUR0211 - Error
### Method marked with the CopyFromMethodAttribute already has a declaration.

## Example

### Code with diagnostic:

```csharp
using Durian;

public partial class Test
{
	// DUR0211
	[CopyFromMethod("Example")]
	public partial void Method();

	public void Example()
	{
	}
}

public partial class Test
{
	public partial void Method()
	{
	}
}

```
