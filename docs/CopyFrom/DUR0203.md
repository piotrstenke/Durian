# DUR0203 - Error
### Target member cannot be resolved.

## Example 1

### Code with diagnostic:

```csharp
using Durian;

public partial class Test
{
	// DUR0203
	[CopyFromMethod("Example")]
	public partial void Method();
}

```

## Example 2

### Code with diagnostic:

```csharp
using Durian;

// DUR0203
[CopyFromType("Other")]
public partial class Test
{
}

```
