v.1.2.0

- Changed the name of the main namespace to Durian.Analysis
- Added support for caching already analyzed members (see: the Durian.Analysis.Cache namespace).
- Added new IDiagnosticReceiver implementations.
- Added new *Filter structs that can enumerate syntax nodes.
- Added new source generator types.
- Added IDurianAnalysisContext - an interface that abstracts out the usage of Analysis- and CompilationStartAnalysis contexts.
- Added some new interfaces that create, filtrate and validate syntax nodes.
- Limited the memory usage of the DurianGenerator* classes.

- Added some keyword-centered methods in AnalysisUtilities.
